== Description ==

X -Pay è il gateway di pagamento CartaSi semplice da integrare su WordPress-WooCommerce per accettare pagamenti online, aperto a tutti i circuiti di pagamento più diffusi e in grado di gestire tutti i canali di vendita (e-commerce, mobile).

Con il gateway di pagamento X-Pay hai a disposizione:

1.	Offerta completa sui metodi di pagamento integrati:
	-	Gestione transazioni sui circuiti internazionali Visa, Mastercard, Visa Electron, V-Pay, Maestro, American Express, Diners;
	-	Accettare vari tipi di Wallet: MySi, Masterpass e PayPal.
2.	Piattaforma multicanale grazie a una pagina di pagamento ottimizzata per mobile, tablet e desktop e alla compatibilità con l’app del merchant tramite pacchetti SDK per Android e IOS.
3.	Interfaccia di pagamento semplice e intuitiva per migliorare l'esperienza d'acquisto dei clienti finali, offrendo diverse possibilità di pagamento online (Easy checkOut, Pagamenti ricorrenti, OneClickPay).
4.	Funzionalità per accettare pagamenti da tutto il mondo:  
	-	I clienti finali possono pagare in euro o in altre valute, con tasso di cambio garantito, grazie al servizio Dynamic Currency Conversion (DCC)
5.	Protezione e sicurezza di tutte le transazioni di pagamento. Utilizzo dei protocolli crittografici SLL-TLS, la procedura 3D Secure e i codici di sicurezza CVV2. Inoltre X-Pay è conforme allo standard PCI-DSS.
6.	Massima flessibilità e personalizzazione:
	-	Logo del sito di e-commere sulla pagina di pagamento;
	-	Pagina di pagamento integrata sul sito senza dover gestire i dati sensibili dei clienti finali grazie a un'interfaccia personalizzabile (iframe o lightbox);
	-	Checkout personalizzato al 100% grazie al servizio Hosted Fields;
	-	Possibilità di scegliere la lingua dell'interfaccia di pagamento in ambiente CartaSi.
7.	Disponibilità di integrazione con l’ambiente di lavoro del merchant:
	-	Configurazione delle modalità di incasso: Immediato, differito, totale o parziale;
	-	Integrazione tramite API con il gestionale merchant per la gestione degli ordini direttamente;
	-	Reportistica avanzata (transazioni, profilazione della clientela, gestione di incassi, annulli e rimborsi, gestione dei contratti per pagamenti ricorrenti);
	-	Business continuity: X-Pay ti garantisce un'operatività costante, senza interruzione del servizio.
8.	Scelta del conto corrente su cui accreditare i pagamenti delle transazioni online.
9.	Assistenza commerciale e tecnica specializzata.

== Installation ==
= Italiano =
1. Assicurati di aver installato l'ultima versione del plugin WooCommerce
2. scompatta e carica i contenuti del plugin nella tua cartella: /wp-content/plugins/
3. Attiva il plugin attraverso il menu "Plugins" in WordPress

= English =
1. Ensure you have latest version of WooCommerce plugin installed
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Configuration ==
1. Visit the WooCommerce settings page, and click on the Payment Gateways tab.
2. Activate the plugin in the WordPress Plugin Manager.
3. Click on CartaSi Ecommerce to edit the settings. 
4. Enable the Payment Method.
5. Fill the field ALIAS, MAC provided by CartaSI.
6. Fill name the title in Credit Card / Debit Card / Internet Banking that will show up on the payment page. 
7. Enable/disable the TEST Mode for testing porpouse, using the fake data supply by CartaSI.
8. Click Save.
